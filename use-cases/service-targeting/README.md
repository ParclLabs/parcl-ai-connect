# Service Targeting: PM Fragmentation

Identify fragmented property management relationships across SFR portfolios to find consolidation opportunities. Portfolios using multiple PM companies are high-value targets for unified property management services.

![PM Fragmentation Demo](../../assets/gifs/property_manager_fragmentation_small.gif)

## What You Get

An interactive HTML dashboard showing:

- PM concentration scoring per portfolio (Single PM → Highly Fragmented)
- Opportunity signal classification (High / Medium / Low-Medium)
- Visual PM mix breakdowns with hover tooltips
- Market cards with drill-down (multi-market mode)

## Choose Your Path

### Basic (Copy & Paste)

Copy the prompt from [`basic/PROMPT.md`](basic/PROMPT.md) into Claude Code. Targets Tampa by default. Change the market name for any metro.

### Advanced (Skill)

Run `/pm-fragmentation [market name]` for a single-market analysis, or `/pm-fragmentation top-10` for a comprehensive multi-market view anchored on the top 10 institutional SFR markets.

See the [skill definition](../../.claude/skills/pm-fragmentation/SKILL.md) for full details.
