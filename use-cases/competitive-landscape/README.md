# Competitive Landscape Analysis

Build comprehensive portfolio teardowns of institutional SFR investors using Parcl Labs Portfolio Hunter. Analyze any investor's geographic footprint, buy box, transaction history, rental pricing strategy, and algorithmic pricing adoption.

![Competitor Analysis Demo](../../assets/gifs/competitor-analysis_small.gif)

## What You Get

A professional Excel workbook with 8 analytical tabs covering:

- Portfolio overview and market concentration
- Rental pricing benchmarked against market rates
- Acquisition vs disposition trends
- Algorithmic pricing detection
- Active listing inventory and disposition signals

## Choose Your Path

### Basic (Copy & Paste)

Copy the prompt from [`basic/PROMPT.md`](basic/PROMPT.md) into Claude Code. Change the investor name to target any institutional SFR operator.

### Advanced (Skill)

Run `/competitor-analysis [investor name]` in Claude Code for a fully parameterized analysis with normalized PM names, comprehensive algo-pricing classification, and market rent benchmarking.

See the [skill definition](../../.claude/skills/competitor-analysis/SKILL.md) for full details.
